🎯 Best Practices

✅ Defer les composants lourds (>100 KB) : PDF, editors, charts
✅ Defer les composants optionnels : comments, below-fold content
✅ Utiliser @placeholder pour éviter layout shift
❌ Ne pas defer le contenu above-the-fold ou critique pour SEO
❌ Ne pas defer les composants légers (<10 KB)
❌ Éviter la sur-optimisation : focusez sur les gros gains
